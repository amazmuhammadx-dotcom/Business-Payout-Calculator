
# Payout Calculator Pro

A free Streamlit MVP for automated salary payout, tax deduction, Pakistan transfer, and PKR distribution calculation.

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to deploy free

1. Create a GitHub account.
2. Create a new public repository.
3. Upload `app.py` and `requirements.txt`.
4. Go to Streamlit Community Cloud.
5. Connect GitHub.
6. Select the repo and deploy `app.py`.

## Main inputs

- Client Name
- Date Range
- Total Salary Before Tax
- Bank Exchange Rate
- Actual PKR Received
- Overall Tax Rate
- Pakistan Business Tax Rate
